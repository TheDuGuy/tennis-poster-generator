## How to Use

### 1. Set Month and Year
- Choose the year and month at the top of the form

### 2. Add Header Information
- Club name
- Title (default: "WHAT'S ON")
- Header info (e.g., "Pricing & membership info: mardensportsclub.com")
- Website URL

### 3. Add Events to Days
- Click on any day number in the mini calendar grid
- Click "Add Event" to create a new event for that day
- Fill in:
  - **Time** (optional) - e.g., "16:00-18:00"
  - **Event title** - e.g., "CLUB NIGHT", "American Tournament"
  - **Event type** - Tournament (bold), Social, Meeting, etc.

### 4. Add Side Panel Notes
- Click "Add Note" in the notes section
- Each note can have:
  - Title (optional)
  - Content (main text)

### 5. Add Additional Info
- **Committee info** - committee members, roles
- **Captains info** - team captains, contacts
- **QR code URL** - your website, WhatsApp group, etc.

### 6. Print or Save as PDF
- Click "Print Poster" button
- Choose "Save as PDF" to create a file
- Or print directly to paper
- **Important:** Make sure to select **LANDSCAPE** orientation in print settings!

## Print Tips

1. **Use landscape orientation** - this is critical!
2. **Enable "Background graphics"** in print dialog
3. **A4 paper size** selected
4. **Margins:** 10mm all around
5. **Preview before printing** - the on-screen preview shows exactly how it will print
